§ IV

Frequently Asked

Is my figure uploaded to a server?
No. All file inspection happens in your browser using JavaScript. Your figure never leaves your computer. We don't have a server that processes uploads — by design.
How accurate are the journal requirements?
Specs are curated from publisher author guidelines and updated quarterly. They're meant as a first-pass sanity check — always verify against the journal's current submission portal, since requirements occasionally change between updates. We are not affiliated with any publisher.
Why doesn't it check line weights or font sizes?
Those checks need server-side parsing of vector files (PDF, EPS) with tools like Ghostscript. We're building it for the Pro tier. The current free version covers the most common rejection reasons: wrong DPI, wrong dimensions, wrong color mode, wrong file format.
My journal isn't listed. Can you add it?
Yes — email us via the waitlist signup with the journal name and a link to its author guidelines. We add requested journals weekly.
Will FigCheck always be free?
The directory and basic checker (format, DPI, dimensions, color mode) will always be free. Advanced features — vector validation, batch processing, auto-fix tools — will be a paid Pro tier when they ship.
